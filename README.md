# Smart Pig

Marketing site for **Smart Pig**, a charity that gives kids real-world entrepreneurship opportunities — workshops, mentorship, and seed grants to launch their first businesses.

## Stack

Single static page — `index.html` with embedded CSS, no build step. Logo assets live in `assets/`.

## Local preview

Open `index.html` in a browser, or run any static server, e.g.:

```
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deploy

The site is deployed on Vercel. Pushing to `main` triggers a new deployment.
